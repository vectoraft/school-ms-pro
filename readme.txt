School MS Pro
=============

A professional, modern school management system plugin for WordPress. Includes full React dashboard, Elementor widgets, and advanced features for Super Admin and School Staff.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/school-ms-pro` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the shortcodes or Elementor widgets to access dashboards and features.

== Features ==
- React + Tailwind CSS dashboard
- Super Admin & School Staff roles
- Analytics, user/role management, transactions, reporting, messaging, notifications, library, audit logs, and more
- Elementor widgets and shortcodes for all dashboards
- Clean uninstall, translation-ready
